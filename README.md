# AI-Assisted Task Management & Adaptive Planning System

## Overview
An AI-powered adaptive planning assistant that converts goals into structured task roadmaps, schedules them into available time blocks, collects execution feedback, and generates intelligent replanning suggestions.

## Features
- Goal Intake & Clarification
- AI Roadmap Generation
- Dependency-Aware Scheduling
- Adaptive Replanning
- Assistant Chat
- Plan History & Revert
- Web Search Integration
- Local Gemma Runtime Support

## Tech Stack

- FastAPI
- SQLite
- HTML/CSS/JavaScript
- Gemma
- Jina AI

- DuckDuckGo
